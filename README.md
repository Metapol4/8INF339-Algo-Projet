# 8INF339-Algo-Projet
## Fonctionalitées
### Général
- "R" pour réinitialiser la grille, le joueur et les ennemis
 ### Recherche de chemin
 - Pointer avec le curseur et appuyer sur "E" pour déplacer le joueur selon l'algorithme de recherche de chemin choisi
 - "Tab" pour changer d'algorithme de recherche de tri (BFS/Dijskstra)
 - Notes sur la grille:
	 - Les cases noires valent 1
	 - Les cases bleues valent 5
	 - Les cases grises valent 10 
	 - Les cases pleines sont des murs, qui ne sont pas inclues dans les adjacences.
 ### Tri
- "Space" pour démarrer la séquence selon le tri choisi
- "Q" pour changer de tri (Valeur, Distance, Distance et Valeur, Aucun)
### Programmation dynamique
- "D" pour démarrer la séquence (utilise l'algorithme de recherche de chemin choisi)
